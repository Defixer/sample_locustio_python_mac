This is a sample locustio script for mac

## Requirements
	*Python
	*Locustio
	*Virtualenv

## Creating virtual environment
```
python -m virtual env
```

## Installing python
```
brew install python
```

## Installing Locustio
```
pip install locustio
``` 
see [Locustio Installation](https://docs.locust.io/en/stable/installation.html)
